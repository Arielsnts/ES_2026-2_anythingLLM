# Auditoria de Maturidade em Ecossistemas LLM  
## Projeto: AnythingLLM

Este repositório contém a auditoria de maturidade em engenharia de software aplicada ao projeto AnythingLLM, desenvolvida como atividade avaliativa da disciplina de Engenharia de Software da Universidade Federal de Sergipe (UFS).

O objetivo é analisar o nível de maturidade do projeto com base nos modelos MPS.BR e CMMI, identificando práticas adotadas, lacunas e propostas de melhoria.

## Metodologia

A análise foi realizada com base em:

- Exploração do repositório do projeto no GitHub (código, issues, pull requests e commits)  
- Leitura da documentação disponível (README, CONTRIBUTING.md)  
- Comparação com os modelos MPS.BR e CMMI  
- Identificação de evidências práticas no projeto
